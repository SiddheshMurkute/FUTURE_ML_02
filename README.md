# 📉 Customer Churn Prediction System (Task 2)

## 🧠 Project Overview
Customer churn is a critical business problem where companies lose customers over time.  
This project builds a **machine learning–based churn prediction system** to identify customers likely to leave a service and enable data-driven retention strategies.

The system analyzes customer behavior and financial attributes to generate **churn risk probabilities** and actionable insights through a **visual dashboard**.

---

## 🎯 Objectives
- Predict customers likely to churn
- Identify high-risk customers using probability scores
- Analyze churn patterns across geography and credit scores
- Support business decision-making using data insights
- Visualize churn trends using an interactive dashboard

---

## 🛠️ Tech Stack
- **Language:** Python  
- **Libraries:** Pandas, NumPy, Scikit-learn, XGBoost  
- **Environment:** Jupyter Notebook  
- **Visualization:** Power BI  
- **Data Format:** CSV

---

## 🔍 Methodology                 
- Data preprocessing and feature engineering
- Exploratory data analysis to uncover churn drivers
- Supervised machine learning using XGBoost
- Probability-based churn risk scoring
- Dashboard visualization for insight analysis

---

## 📈 Dashboard Insights
The dashboard provides:
- Overview of total and at-risk customers
- Churn distribution by country
- Credit score impact on churn probability
- Customer-level churn risk percentages

A dashboard screenshot and .pbix file is included in the repository.

---

## 🚀 Key Learnings
- End-to-end machine learning pipeline design
- Gradient boosting (XGBoost) for churn prediction
- Translating ML outputs into business insights
- Data-driven decision support using dashboards
